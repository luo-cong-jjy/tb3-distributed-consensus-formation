# 激光雷达测试工具

基于 `tb3_controller_node.py` 的激光雷达处理逻辑创建的独立测试脚本，用于测试任意TurtleBot3机器人的激光雷达是否工作正常。

## 功能特性

- 🔍 **实时监测**: 持续监听激光雷达数据并计算障碍物信息
- 📊 **统计分析**: 显示回调频率、点数统计、距离分析等
- ⚙️ **灵活配置**: 支持指定机器人ID或直接指定话题名称
- 🚨 **健康检查**: 自动检测激光雷达工作状态并给出警告
- 📈 **历史记录**: 记录最近距离历史并计算统计数据

## 使用方法

### 基本用法

```bash
# 测试 tb3_0 的激光雷达（默认）
python3 laser_test.py

# 测试指定机器人的激光雷达
python3 laser_test.py --robot_id 1    # 测试 tb3_1
python3 laser_test.py --robot_id 2    # 测试 tb3_2
```

### 高级用法

```bash
# 直接指定激光雷达话题
python3 laser_test.py --topic /tb3_3/scan

# 自定义检测范围和输出间隔
python3 laser_test.py --robot_id 0 --range 3.0 --interval 0.5

# 指定自定义里程计话题
python3 laser_test.py --topic /custom_robot/scan --odom_topic /custom_robot/odom
```

### 参数说明

- `--robot_id`: 机器人ID (0-4)，自动生成标准话题名称
- `--topic`: 直接指定激光雷达话题名称
- `--odom_topic`: 里程计话题名称（可选，会自动推断）
- `--range`: 障碍物检测最大距离（米，默认2.0）
- `--interval`: 统计信息输出间隔（秒，默认1.0）
- `--help`: 显示帮助信息

### 健康状态指示

- ✅ **激光雷达工作正常**: 回调频率在5-30Hz之间
- ⚠️ **警告: 回调频率较低**: 可能存在通信问题或数据丢失
- ⚠️ **警告: 回调频率过高**: 可能影响系统性能
- ❌ **错误: 未收到数据**: 激光雷达话题不存在或无数据发布

## 使用场景

1. **启动前检查**: 在运行分布式控制前，验证所有机器人的激光雷达工作正常
2. **故障诊断**: 当避障行为异常时，检查激光雷达数据质量
3. **性能调优**: 监测激光雷达频率和数据量，优化系统性能
4. **环境测试**: 在不同环境下测试激光雷达的检测效果

## 技术细节

- 基于与 `tb3_controller_node.py` 相同的激光雷达处理逻辑
- 使用相同的坐标变换算法（极坐标→本体坐标→全局坐标）
- 支持与控制脚本相同的检测范围和参数配置
- 提供详细的统计分析和健康检查功能
